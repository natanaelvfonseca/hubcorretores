# HUB Corretores do Litoral SC

Plataforma proprietária para organizar a comunidade imobiliária do litoral catarinense em um ambiente premium de networking, oportunidades, diretórios, benefícios e governança.

## Nova direção do produto

- comunidade digital proprietária
- ecossistema imobiliário regional
- central de networking e geração de negócios
- ambiente premium para corretores, imobiliárias, construtoras e parceiros
- WhatsApp como apoio operacional, não como estrutura principal

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- React Router

## Executar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
